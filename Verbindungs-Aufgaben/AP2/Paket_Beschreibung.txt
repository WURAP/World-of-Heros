Arbeitspaket 2: Bluetoothverbindung Pc-Cyton + Datenübertragung

Anfängliche Recherche Arbeiten:

-) worum es sich bei den Boards handelt

-) wie auf Bluetooth zugegriffen werden kann

-) wie man auf einen USB Port zugreifen kann ( Für das Bluetooth Dongle)

-) Programmier Tutorials für Java in Eclipse, für Arduino Boards

-) Gui's erzeugen um Ergebnisse anzeigen zu können



Erste Programmierfortschritte und Fehler:

-) Anfängliche Problemstellung, wie auf Bluetooth von Cyton über Java zugegriffen werden kann

-) Kompatiblitätsprobleme mit verschiedenen Betriebssystemen ( bestehend bis zum Ende!)

-) GUI Erzeugung für Verbindungsbutton und Test-Anzeigegraphen erfolgreich jedoch ohne Testdaten nicht wirklich zu gebrauchen

-) Port über Java “geöffnet” jedoch trotzdem keine Daten von Cyton erhalten

-) theoretische Lösung für Abspeicherung der Daten als csv-File durch extra Klasse geschafft.

-) Fehlerfreier Code für “Verbindung herstellen” und anzeigen, jedoch weiterhin kein Erhalt von Daten durch die Bluetooth Verbindung

-)Herausgefunden, dass Daten in viel komplexerer Weise übertragen werden als Anfangs angenommen. Benötigt decodierung und umwandlung von ASCI-Code in Volt Angaben.



Zusammenfassung Output zu Lehreinheit 10:

Fehlerfreier Javacode zur “theoretischen” Darstellung und Speicherung eines Datenstreams, welcher über eine Bluetooth Verbindung von Cyton-Board zu PC gelangt. 
USB Port kann angesprochen und geöffnet werden (Betriebssystem abhängig), jedoch keine intakte Bluetooth Verbindung hergestellt werden. 
End-Daten werden theoretisch in einem CSV-File abgespeichert und exportiert, durch eigens erstellte Hilfsklasse (CsvWriter.java). 
Die vollständige funktionierende Verbindung herzustellen und die Daten erfolgreich und vollständig zu übertragen übersteigen meine Programmierfähigkeiten.
